提示-Alert

位于页面最顶部导航栏下，提供三种不同的场景

场景一：成功提示

成功提示在3s后自动消失

您的操作已经成功。3s后此提示自动消失。

代码示例 

    
      //html
      <button class="btn px-btn px-btn-success" (click)="showSuccess()">打开成功提示</button>
      <div class="px-alert px-alert-success" *ngIf="isShowSuccess">
        您的操作已经成功。3s后此提示自动消失。
      </div>

      //typescript
      isShowSuccess:boolean = false;
      showSuccess() {
        this.isShowSuccess = true;
        setTimeout(()=> {
          this.isShowSuccess = false;
        }, 3000);
      }

场景二：警告提示

警告提示需要手动关闭

警告！系统存在安全风险，请进行检测。

代码示例 

    
      //html
      <button class="btn px-btn px-btn-default" (click)="isShowAttention = true">打开警告提示</button>
        <div class="px-alert px-alert-attention" *ngIf="isShowAttention">
        <a role="button" class="px-alert-close" (click)="isShowAttention = false"></a>
        <b>警告！</b>系统存在安全风险，请进行检测。
      </div>

      //typescript
      isShowAttention:boolean = false;
