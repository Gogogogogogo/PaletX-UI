输入框-Input

提供两种不同尺寸的输入框，包含内容说明，验证样式

场景一：基本类型

代码示例 

    
      <input class="px-input lg" placeholder="大尺寸">
      <input class="px-input sm" placeholder="小尺寸">

场景二：禁用的

代码示例 

    
      <input class="px-input lg" placeholder="禁用的">
      <input class="px-input sm" placeholder="禁用的">

场景三：带有说明文字的

内容说明：只能包含字母、数字及下划线

代码示例 

    
      <div class="px-input-wrapper">
        <input class="px-input lg" placeholder="带有说明文字的">
        <span>内容说明：只能包含字母、数字及下划线</span>
      </div>

场景四：带有校验的

输入不能为空

代码示例 

    
      //html
      <div class="px-input-wrapper">
        <input class="px-input lg" placeholder="带有校验的" required [(ngModel)]="inputValue" #input="ngModel">
        <span class="validation" *ngIf="input.errors?.required">输入不能为空</span>
      </div>

      //typescript
      inputValue:string;

场景五：带有单位或图标的

GB
GB

代码示例 

    
      <div class="px-input-wrapper">
        <input class="px-input lg addon" placeholder="带有单位或图标的">

      </div>
      <div class="px-input-wrapper">
        <input class="px-input sm addon" placeholder="带有单位或图标的">
