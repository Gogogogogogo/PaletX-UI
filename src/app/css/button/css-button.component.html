按钮-Buttons

提供四类不同场景的按钮

场景一：标题旁的按钮

单个操作

代码示例 

    
      <button class="btn px-btn px-btn-default">单个操作</button>
      <button class="btn px-btn px-btn-default" disabled>单个操作</button>

多个操作

代码示例 

    
      <div ngbDropdown>
        <button class="btn px-btn px-btn-default px-btn-default-more" ngbDropdownToggle>更多操作</button>
        <div class="px-btn-default-more-dropdown-menu dropdown-menu">
          <button class="btn px-btn px-btn-default">操作1</button>
          <button class="btn px-btn px-btn-default">操作2</button>
          <button class="btn px-btn px-btn-default">操作3</button>
        </div>
      </div>

场景二：常用按钮

代码示例 

    
      <button class="btn px-btn px-btn-success">常用按钮</button>
      <button class="btn px-btn px-btn-success create">创建按钮</button>

场景三：表格按钮

单个操作

代码示例 

    
      <button class="btn px-btn px-btn-operate">编辑</button>
      <button class="btn px-btn px-btn-operate" disabled>编辑</button>

多个操作

代码示例 

    
      <div ngbDropdown>
        <button class="btn px-btn px-btn-operate px-btn-operate-more">编辑</button>
        <button class="btn px-btn px-btn-operate-more-dropdown-toggle" ngbDropdownToggle></button>
        <div class="px-btn-operate-more-dropdown-menu dropdown-menu">
          <button class="btn px-btn px-btn-operate">下载</button>
          <button class="btn px-btn px-btn-operate">删除</button>
          <button class="btn px-btn px-btn-operate">其他</button>
        </div>
      </div>

场景四：组合按钮

确认取消按钮

代码示例 

    
      <button class="btn px-btn px-btn-primary">确认</button>
      <button class="btn px-btn px-btn-cancel">取消</button>
      <button class="btn px-btn px-btn-primary" disabled>确认</button>
      <button class="btn px-btn px-btn-cancel" disabled>取消</button>

删除取消按钮

代码示例 

    
      <button class="btn px-btn px-btn-delete">删除</button>
      <button class="btn px-btn px-btn-cancel">取消</button>
      <button class="btn px-btn px-btn-delete" disabled>删除</button>
      <button class="btn px-btn px-btn-cancel" disabled>取消</button>
