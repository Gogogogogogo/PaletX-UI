模态框-Modal

模态框提供两种不同的场景

场景一：确认提交模态框

代码示例 

    
      //html
      <button class="px-btn px-btn-primary" (click)="open1()">打开确认提交模态框</button>
      <template #modal1_template>
        <div class="px-modal-header">
          <h4>确认</h4>
          <a role="button" (click)="modal1.close()"></a>
        </div>
        <div class="px-modal-body">
          <div class="px-modal-confirm">
            <img src="../../../../public/images/modal_confirm.svg">
            <h5>确认提交吗？</h5>
            <small>提交后可以查看提交数据</small>
          </div>
        </div>
        <div class="px-modal-footer">
          <button class="px-btn px-btn-primary">确认</button>
          <button class="px-btn px-btn-cancel" (click)="modal1.close()">关闭</button>
        </div>
      </template>
      
      //typescript
      @ViewChild('modal1_template') modal1_template:any;
      modal1:any;

      constructor(private modalService:NgbModal) {
      }

      open1() {
        this.modal1 = this.modalService.open(this.modal1_template, {
          backdrop: true,
          keyboard: true
        });
      }
